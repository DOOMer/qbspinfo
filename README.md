QBspInfo
==========

version 0.5

App for getting info about Quake 3 BSP files.

Build requirements
------------------

 * Qt5 >= 5.0 (Qt 4.x isn't support)
 * CMake >= 2.6 (only for build)
 * GCC >= 4.6 (pnlu for build)

Build
-----

Build ScreenGrab, using these commands:

	mkdir build
	cd build
	cmake [BUILD OPTIONS] ../
	make
	make install
